# SketchSprint 🎨

A real-time multiplayer drawing and guessing game - a modern clone of Skribbl.io built from scratch.

**[Play Live →](https://skribbl.amitesh.work/)**

## Features

- Create and join game rooms
- Real-time drawing with live sync
- Chat-based word guessing
- Dynamic scoring system
- Complete game flow with winners

## Tech Stack

- **Next.js** + **TypeScript**
- **Bun** runtime
- **Raw WebSockets** for real-time communication

## Why This Project?

Built as a proof of work to demonstrate full-stack skills and real-time multiplayer implementation from first principles. No game frameworks - just pure WebSocket magic.

*Inspired by [skribbl.io](https://skribbl.io/)*
